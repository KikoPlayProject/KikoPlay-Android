# KikoPlay新版Android App需求与页面交互设计
[KikoPlay](https://kikoplay.fun/about/)是一款PC弹幕播放器，它提供了局域网服务功能（具体接口见@kikoplay_lan.md），有一个[android端](https://github.com/KikoPlayProject/KikoPlay-Android-LAN)，可以在手机上访问KikoPlay PC端的播放列表，播放视频并加载弹幕。

这个android app上次更新是五年前了，现在看起来有些简陋，现在我想重新做一个。我的想法是：

- 包含现有app的全部功能：读取pc端的播放列表，可以播放视频，加载弹幕/字幕，另外远程截图、截取片段、调整弹幕延迟/时间轴这些功能也保留
- 可以将pc端的条目缓存到手机，这样就算不在同一个局域网，用手机还能继续看
- 可以列出、播放手机本地存的视频，识别文件并加载弹幕。识别功能通过KikoPlay的服务KService进行（与服务器的数据交互格式定义在[service.proto](https://github.com/KikoPlayProject/KikoPlay/blob/master/Service/pb/service.proto)，具体用法参考[代码](https://github.com/KikoPlayProject/KikoPlay/blob/master/Service/kservice.h和https://github.com/KikoPlayProject/KikoPlay/blob/master/Service/kservice.cpp)，服务器只能返回弹幕源，具体加载弹幕的逻辑也需要实现）
- 整体界面更加美观、现代化

页面功能与交互详细拆解

## 一级页 1：首页
 - 页面定位：用户入口页，聚合所有高频功能
 - 页面结构：PC 播放列表快捷卡片 + 最近观看模块 + 底部导航
 - 核心组件与交互：
   - PC 播放列表快捷卡片
     - 组件：卡片容器 + PC 名称 + "查看全部" 按钮 + 最近 4 个目录列表
     - 交互：点击卡片任意位置进入 PC 播放列表根目录；点击单个目录直接进入对应目录；如果未连接，点击进入局域网连接页
   - 最近观看模块
     - 组件：模块标题 + 右上角「更多」按钮 + 横向滚动列表
     - 列表条目：缩略图 + 标题 + 播放进度条 + 设备来源图标（PC / 手机）
     - 交互：点击条目直接跳转到对应播放进度；点击「更多」进入完整播放历史页
  - 状态说明：
    - 未连接 PC：PC 播放列表快捷卡片为空，显示“连接PC”
    - 无最近观看：显示空状态插画 + "还没有观看记录" 文字
 - 跳转关系：
   - 可跳转到：局域网连接页、完整播放历史页、视频播放页
   - 可从：所有一级页返回
## 一级页 2：本地视频页
 - 页面定位：手机本地视频管理与弹幕智能识别
 - 页面结构：顶部筛选栏 + 内容区 + 批量操作栏（长按触发）
 - 核心组件与交互：
   - 顶部筛选栏
     - 左侧：视图切换按钮（列表 / 文件夹）→ 点击切换视图模式
     - 右侧：排序按钮（名称 / 修改时间 / 大小）→ 点击弹出排序选项
     - 右侧：筛选按钮 → 点击弹出筛选选项（全部 / 未识别 / 已识别）
   - 内容区
     - 列表视图：
       - 条目：缩略图 + 文件名 + 时长 + 文件大小 + 右侧识别状态标识
       - 交互：点击条目进入播放页；点击识别失败的红色感叹号重新识别
     - 文件夹视图：
       - 条目：文件夹图标 + 文件夹名称 + 视频数量
       - 交互：点击文件夹进入子目录；点击视频文件进入播放页
   - 批量操作栏（长按任意条目触发）
     - 组件：全选按钮 + 已选数量 + 操作按钮（识别弹幕 / 取消识别 / 删除）
     - 交互：点击操作按钮批量执行对应功能
 - 状态说明：
   - 未授权存储权限：显示空状态插画 + "请授权存储权限" + 授权按钮
无本地视频：显示空状态插画 + "本地没有找到视频文件"
   - 识别中：显示顶部进度条 + "正在识别 X 个视频"
 - 跳转关系：
  - 可跳转到：视频播放页
  - 可从：首页返回
## 一级页 3：缓存管理页
 - 页面定位：PC 端视频缓存任务与已缓存内容管理
 - 页面结构：顶部标签栏 + 内容区 + 存储空间统计条
 - 核心组件与交互：
   - 顶部标签栏
     - 标签 1：缓存中 → 显示正在缓存和等待缓存的任务
     - 标签 2：已缓存 → 显示已缓存完成的视频
   - 缓存中页面
     - 任务条目：缩略图 + 标题 + 进度条 + 速度 / 剩余时间 + 暂停 / 继续按钮 + 取消按钮
     - 交互：长按条目可拖动调整任务优先级；点击暂停 / 继续按钮控制任务
     - 顶部：全部暂停 / 全部开始按钮
   - 已缓存页面
     - 视频条目：缩略图 + 标题 + 缓存大小 + 缓存时间 + 播放进度条
     - 交互：点击条目进入播放页；长按弹出菜单（同步进度到 PC / 删除 / 重新缓存）
   - 存储空间统计条
     - 组件：进度条 + "已用 XGB / 共 YGB" 文字
     - 交互：点击进度条直接进入手机系统存储设置
 - 状态说明：
   - 无缓存任务：显示空状态插画 + "没有正在缓存的任务"
   - 无已缓存视频：显示空状态插画 + "还没有缓存任何视频"
 - 跳转关系：
   - 可跳转到：视频播放页
   - 可从：首页返回
## 一级页 4：设置页
 - 页面定位：应用全局配置与账号管理
 - 页面结构：分组列表布局
 - 核心组件与交互：
   - 账号管理组
     - 未登录："未登录" + 箭头 → 点击进入登陆页
     - 已登录：用户名 + 头像 + 箭头，增加退出登陆按钮
   - 缓存设置组
     - 缓存路径：当前路径 + 箭头 → 点击弹出路径选择器
     - 最大缓存空间：当前值 + 箭头 → 点击弹出空间选择器（1GB/5GB/10GB / 无限制）
     - 自动清除过期缓存：开关按钮（默认关闭）
   - 同步播放进度：开关按钮（默认开启）
   - 关于与帮助组
     - 检查更新："当前版本 X.X.X" + 箭头
     - 官方网站：文字 + 箭头 → 点击打开浏览器
     - 反馈建议：文字 + 箭头 → 点击打开邮件应用
     - 开源协议：文字 + 箭头 → 点击进入协议详情页
  - 跳转关系：
    - 可跳转到：登陆页、协议详情页
    - 可从：首页返回
### 二级页 1：局域网连接页（LAN Connection）
 - 页面定位：连接 PC 端 KikoPlay 服务
 - 页面结构：自动扫描区 + 手动连接区 + 历史记录区 + 帮助区
 - 核心组件与交互：
   - 自动扫描区
     - 组件：扫描动画 + "正在扫描局域网设备..." 文字
     - 发现设备列表：设备名称 + IP 地址 + 连接按钮
     - 交互：点击连接按钮开始连接；连接成功自动跳转到 PC 播放列表根目录
   - 手动连接区
     - 组件：输入框（提示 "请输入 PC 端服务地址，如 192.168.1.100:8080"） + 连接按钮
     - 交互：输入地址后点击连接按钮
   - 历史连接记录区
     - 组件："历史连接" 标题 + 记录列表
     - 记录条目：设备名称 + IP 地址 + 连接时间
     - 交互：点击条目快速重连；左滑条目显示删除按钮
   - 帮助区
     - 组件："如何开启 PC 端服务？" + 展开按钮
     - 交互：点击展开显示图文步骤
  - 状态说明：
   - 连接中：显示加载动画 + "正在连接..."
   - 连接失败：弹出 Toast 提示 "连接失败，请检查 PC 端服务是否开启" + 重试按钮
  - 跳转关系：
   - 可跳转到：PC 端播放列表页
   - 可从：首页返回
### 二级页 2：PC 端播放列表页（PC Playlist）
 - 页面定位：展示并操作 PC 端同步的树形播放列表
 - 页面结构：顶部导航栏 + 内容区 + 批量操作栏（长按触发）
 - 核心组件与交互：
   - 顶部导航栏
     - 左侧：返回按钮
     - 中间：路径下拉菜单（显示当前完整路径，如 "动画 / 2026 年 4 月 / 某某番"）
     - 右侧：筛选按钮
     - 交互：点击路径下拉菜单展开目录树，可快速跳转到任意上级目录；点击筛选按钮弹出筛选选项（全部 / 未播放 / 未看完 / 已看完）
   - 内容区
     - 目录条目：文件夹图标 + 目录名称 + 子节点数量 + 展开 / 收起箭头
     - 视频条目：左侧缓存状态图标  + 标题 + 播放状态 + 时长
     - 交互：点击目录进入子目录；点击视频条目进入播放页；点击缓存图标触发缓存操作
     - 批量操作栏（长按任意条目触发）
       - 组件：全选按钮 + 已选数量 + 操作按钮（缓存 / 标记已看 / 修改颜色 / 删除）
   - 状态说明：
     - 未连接 PC：显示空状态插画 + "请先连接 PC 端 KikoPlay 服务" + 连接按钮
     - 目录为空：显示空状态插画 + "该目录下没有视频"
  - 跳转关系：
    - 可跳转到：视频播放页、子目录页
    - 可从：局域网连接页、上级目录页返回
### 二级页 3：视频播放页（Player）
 - 页面定位：核心播放页面，整合视频、弹幕、字幕与所有操作
 - 页面结构：播放页分两个形态：竖屏和横屏。竖屏状态下包括上方的播放器和下方的内容信息，横屏状态只有播放器。其中播放器包括 视频渲染层 + 弹幕渲染层 + 顶部状态栏 + 底部控制栏 + 右侧设置面板；竖屏状态下内容信息包括 当前正在播放的文件名、同目录下的所有视频列表及弹幕池信息。
 - 核心组件与交互：
   - 视频播放器
     - 视频渲染层：ExoPlayer/libmpv 内核
     - 弹幕渲染层：DanmakuFlameMaster实现
     - 顶部状态栏
       - 左侧：返回按钮
       - 中间：视频标题
       - 右侧：截图按钮 + 截取片段按钮
     - 底部控制栏
       - 左侧： 播放/暂停按钮 + 弹幕开关
       - 中间：进度条（支持拖拽）+ 当前时间 / 总时长
       - 右侧：发送弹幕按钮（未全屏时隐藏） + 设置按钮（未全屏时隐藏） + 全屏按钮
     - 设置面板：全屏状态下，点击设置按钮才显示。包括：弹幕透明度滑块、字体大小滑块、显示区域选择、弹幕类型过滤、弹幕屏蔽规则、播放速度、字幕相关设置等
     - 手势操作
       - 左侧上下滑动：调节亮度
       - 右侧上下滑动：调节音量
       - 左右滑动：调整进度
       - 双击：播放 / 暂停
       - 单指点击：显示 / 隐藏控制栏
   - 内容信息
     - 上方：当前播放的动画名、剧集名（如果没有就显示文件名）
     - 下方：两个分页，分别显示当前正在播放文件所在目录的全部视频、当前播放视频的弹幕池信息（支持刷新、显示每个源、为每个源设置延迟和时间轴调整）
   - 状态说明：
     - 加载中：显示加载动画 + "正在加载视频..."
     - 播放失败：显示错误插画 + "视频加载失败" + 重试按钮
     - 弹幕加载中：显示小转圈图标在弹幕开关旁，加载失败可以重试
 - 跳转关系：
   - 可跳转到：所有弹窗页面
   - 可从：首页、本地视频页、缓存管理页、PC 播放列表页、完整播放历史页返回
